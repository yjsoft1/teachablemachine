미들웨어

당사가 개발한 미들웨어는 디바이스의 제어 및 포털, 웹어플리케이션 데이터 서비스 등의 모든 메시징 서비스를 솔루션을 갖춘 소프트웨어입니다. 닷넷 플랫폼으로 개발되었으며, 기본적으로 모든 디바이스들을 IP와 PORT기반으로 관리하지만 시리얼 통신의 관리도 가능합니다. 이는 운영되고 있는 서비스단과 운영체제 사이에 위치한 소프트웨어로 비동기소켓 컴포넌트, 데이터베이스 관리 컴포넌트, WEB SERVICE 컴포넌트(JSON, SOAP), 디바이스 관리 컴포넌트 등으로 구성되어져 있습니다. 여러 형태의 컴포넌트들을 갖추고 있으며 수많은  테스트로 인한 안정된 데이터 통신을 보장합니다. 

  1. WEB SERVER - IIS호스팅을 자체적으로 구성하는 웹서버 솔루션입니다. 닷넷플랫폼의 WCF 서비스와 함께 사용이 가능하며, JSON등의 서비스도 제공됩니다.
  2. WEB APPLICATION SERVER - 닷넷 프레임워크를 이용한 클라우드 지원 웹 어플리케이션 서버 플랫폼으로 구성되어 있습니다.
  3. IP, PORT기반의 디바이스 제어 - TCP연결로 관리되는 어떠한 디바이스도 제어 및 관리가 가능합니다.
  4. 시리얼 통신 디바이스 제어 - 시리얼 통신으로 관리되는 어떠한 디바이스도 제어 및 관리가 가능합니다.

시스템 구성

미들웨어의 구성은 TCP/IP통신과 시리얼 통신을 이용한 디바이스를 제어하는 클라이언트 제어부분, 자체 개발된 GIS 지도 플랫폼, 웹 어플리케이션 구동을 위한 WEB SERVICE와 관련된 통신 플랫폼인 WCF, SOAP, JSON을 지원합니다.
