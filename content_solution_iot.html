자체 개발된 GIS 엔진과
안정성 있는 비동기 소켓서버로 안정화된 시스템을 제공합니다.

아이오티뷰

시스템 개요

아이오티뷰는 TCP/IP를 기반으로 접속되는 모든 디바이스의 상태정보를 모니터링 및 제어가 가능하도록 구성되어진 시스템입니다. 자체 개발된 IOCP기반의 소켓 컴포넌트는 모든 기종의 디바이스를 IP/PORT를 기분으로 관리하기 때문에 TCP/IP를 기반으로 접속되는 모든 디바이스의 관리가 가능합니다. 아이오티뷰로 접속되는 디바이스는 하나의 TAG형태로 관리되며 각 디바이스 별 상황정보를 자체개발된 2D GIS 컴포넌트로부터 편리하게 확인할 수 있도록 구성되어 있습니다. 디바이스의 이미지를 사용자가 쉽게 변경이 가능하며 안정성있는 소켓서버와 2D GIS 엔진과의 연동을 통하여 각 디바이스들을 시각적으로 편리하게 모니터링이 가능한 시스템 입니다.

  • GIS

    - GIS 지도의 확대, 축소, 이동, 전체보기

    - 각 디바이스의 드래그 앤 드랍을 통한 위치정보 변경

    - 각 디바이스의 이미지 변경 및 크기 변경

    - GIS 지도를 대처할 수 있는 백그라운드 이미지 등록 기능

    - GIS Layer Color, Visible, LineType, LineThickness 등의 레이어 관리

  • 다비이스 관리

    - GIS 화면에서 디바이스의 추가, 삭제, 검색, 편집 기능

    - 리스트 형태의 화면에서 디바이스의 추가, 삭제, 검색, 편집 기능

    - 디바이스의 접속 상태 디스플레이

    - RFID 리더기의 제어

    - 경광등 제어

    - 각 디바이스 별 실시간 데이터 통신 패킷 디스플레이

  • Network Packet

    - 실시간 네트워크 패킷 디스플레이

    - 각 디바이스의 헬스체크 패킷 디스플레이

    - Ascii 및 Hex 값의 실시간 패킷 확인

    - 실시간 패킷 정보의 엑셀 및 텍스트 내보내기 기능

    - 선택된 디바이스의 실시간 패킷 디스플레이

시스템 특성

윈도우 운영체제에서 운영이 가능한 시스템입니다.

OS Windows 7 ~
프레임워크 .NetFramework 4.5
C# WPF

시스템 구성

이더넷 연결을 기본으로 구성하여 IP/PORT로 이루어진 모든 클라이언트들을 사전에 등록하여, 사전에 설계되어진 특정 프로토콜 패킷을 주고 받을 경우 상태정보 및 제어가 가능하기 때문에 클라이언트의 기종에 상관없이 모두 모니터링이 가능한 시스템 입니다.

  • IP, PORT 관리

    어떠한 클라이언트라도 IP, PORT로 접속이 가능하다면 관리가 가능합니다.

  • 클라이언트 제어

    현장에 설치되어 있는 경광등 및 RFID 및 기타 디바이스에 대해 자동 및 수동 제어가 가능합니다.

  • GIS VIEW

    자체 개발된 GIS 엔진위에 각 클라이언트들의 아이콘을 손쉽게 이동 및 관리가 가능합니다.

  • DBMS

    기본적으로 무료 데이터베이스인 SQLite를 사용하고 있기 때문에 경제적으로 부담이 되지 않는 시스템입니다. 아이오티뷰는 각 기능별로 컴포넌트화하여 개발되어졌기 때문에 사용자의 요청에 따라 다른 데이터베이스로의 변경이 용이합니다.
